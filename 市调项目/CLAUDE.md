# 市调项目 — 市场调研框架搭建

> 正文必须用中文。本文件是项目迁移记忆文件，用于新 Claude Code 环境快速理解项目。

## 项目定位

从"品类调研报告"项目中提炼方法论，搭建**通用化市场调研报告模板框架**。当前以美区汽车玻璃为示例数据填充。

## 模板功能（品类调研报告模板.html）

- 侧边栏导航（11+ 章节）
- KPI 指标卡片
- 标签页切换（概览/详情/对比）
- 响应式布局
- 通用化设计（可替换为任意品类数据）
