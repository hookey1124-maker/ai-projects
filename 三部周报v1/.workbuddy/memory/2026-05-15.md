## v3 HTML JS 语法错误修复

### 问题
浏览器打开 v3 HTML 报错：
1. `SyntaxError: Unexpected token ')'` → 整段 `<script>` 无法解析
2. `ReferenceError: switchTab is not defined` → Tab 切换失效

### 根因（两个独立的语法错误）

**错误1：多行字符串语法**（汇报输出 Tab 的周报文案变量）
- Python f-string 中 `\n` 被解析为真实换行符，导致 JS 中单引号字符串跨行
- 影响 `block1` ~ `block5` 五个变量
- 修复：改为反引号模板字面量 + `\\n` 转义

**错误2：括号不匹配**（Tab1 品线维度表格，HTML 行 7624）
- 表达式 `Math.round(r.curSalesQty > 0 ? (r.curSku - (cum43Data.filter(...)).length)) : 0)` 多了一个 `)`
- 根因：`(cum43Data.filter(...))` 外层多包了一组不必要的括号
- 修复：去掉外层多余括号 → `(r.curSku - cum43Data.filter(...).length)`

### 验证
使用 `new vm.Script(js)` 验证整个 JS 解析通过，无语法错误。

- `新品板块_4.30-5.6_v3.html`（两份同步）
