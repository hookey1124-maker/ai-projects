# 模块验收检查清单

模块交付回来后逐项检查：

- [ ] 是否只修改了自己的模块目录。
- [ ] 是否没有改 DataCenter 核心文件。
- [ ] 是否没有改 Sidebar。
- [ ] 是否没有改全局 CSS。
- [ ] 是否没有直接在页面里解析 Excel。
- [ ] 是否从 `modulePeriodConfig` 获取周期口径。
- [ ] 是否从 `weeklyReportContext` 获取模块数据周期。
- [ ] 是否展示了数据周期和周期口径。
- [ ] 是否展示了数据来源状态。
- [ ] 是否保留空数据状态。
- [ ] 是否只在同周期类型内做环比。
- [ ] 页面内容、指标、图表、表格、异常和汇报输出是否由模块负责人按模块目标自行设计。
- [ ] 是否没有强行套用其他模块的页面结构。
- [ ] `npm run build` 是否通过。
- [ ] 页面是否能从 Sidebar 正常点击进入。
- [ ] 是否影响其他模块。
- [ ] README 是否更新。
- [ ] Adapter 输入输出类型是否清晰。
- [ ] Summary 文案是否面向业务汇报。
