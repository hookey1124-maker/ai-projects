## v3.2：Emoji、动画与可视化图表增强

### 1. CSS 动画

添加了 8 组 `@keyframes` 动画：

| 动画名 | 用途 |
|---|---|
| `fadeInUp` | KPI 卡片渐进上浮入场 |
| `fadeIn` | 通用淡入 |
| `pulse` | 核心指标脉冲呼吸效果 |
| `shimmer` | 表头微光扫过 |
| `slideInLeft` / `slideInRight` | 图表左右滑入 |
| `bounceIn` | 高亮数值弹跳入场 |
| `gradientShift` | 渐变色流动背景 |

### 2. Emoji 点缀

在所有 KPI 标签、图表标题、Section 标题、Tab 名称中加入对应 emoji：
- 📦 SKU · 🛒 销量 · 💰 销售额 · 🎯 出单率 · ⏱ 及时率
- ✅ PLG同开 · 🟠 单PLG · 🔴 单PLG且未出单
- 📈 趋势 · 🥧 占比 · 📊 分布

### 3. 新增 5 个 Chart.js 图表（共 9 个）

**Tab 1 - 核心看板：**
- `chartExpandRate` — 拓展类型出单率对比（柱状图）

**Tab 2 - 未出单/低市占：**
- `chartLowShareCat` — 低市占产品品类分布（环形图）
- `chartUnsoldReason` — 未出单原因分布（柱状图）

**Tab 3 - PLG/广告：**
- `chartPlgClass` — PLG 等级分布（环形图）
- `chartAdMetrics` — 广告核心指标对比 ROAS/CVR/CTR/ACOS（柱状图）

Tab 2 和 Tab 3 的图表采用**懒加载**，首次切换到对应 Tab 时才初始化，保证首页加载性能。

## v3.3：表格滚动容器与粘性表头/表尾

### 1. 表格滚动容器

两个明细表（PLP 广告明细、四三累计明细）添加滚动容器：

```css
.table-scroll-wrap {
  max-height: 68vh;
  min-height: 360px;
  overflow-y: auto;
  overflow-x: auto;
  border: 1px solid #e0e0e0;
  border-radius: 6px;
}
```

### 2. 双端粘性定位

- **表头粘性**：`thead th { position: sticky; top: 0; z-index: 10; }`
- **合计行粘性**：`tfoot td { position: sticky; bottom: 0; z-index: 10; background: #f0f6ff; }`

四三累计表格将合计行从 tbody 迁移到 tfoot，实现滚动时表头、表尾同时冻结。

### 3. PLP 字体回调

字体从 10px 恢复至 `font-size: 12px`，表头/单元格 padding 恢复正常。

### 4. KPI 卡片紧凑化

```css
.kpi-grid { grid-template-columns: repeat(auto-fit, minmax(150px, 200px)); }
```
将 max 从 `1fr` 改为 `200px`，避免卡片在宽屏下过度拉伸。

### 5. 筛选栏像素泄漏修复

```css
.filter-bar-sticky { border-radius: 0; top: -1px; }
```
去除粘性筛选栏的圆角并用 `top: -1px` 向上覆盖 1px，消除冻结栏下方的像素缝隙。
