# 新品月报看板 SOP

## 一、源数据要求

### Excel 文件：`新品月报数据.xlsx`

| Sheet 名称 | 用途 | 关键列 |
|-----------|------|--------|
| 近三月新品明细数据 | 4-5月SKU级明细（157行×61列） | SKU, 上架日期, 分析人, 品类, 4-5月销量/销售额/对手/市占/PLP/PLG |
| 过往产品销售数据 | 1-5月历史SKU级数据（157行×36列） | SKU, 1-5月销量/销售额/对手/市占/分析人 |
| 三部月度汇总 | 部门总数据（1244行×20列） | 每行一个SKU的1-5月销量/销售额/对手/市占，用于自己求和算部门总计 |
| 三部维度汇总 | 品线+分析人维度汇总（34行×17列） | 左半=品线1-5月销量，右半=分析人1-5月销量（只取第一段销量数据，后面是销售额和市占重复段） |
| 5.31PW | PW爬虫近7日快照（1161行×5列） | 销售编号, 对手销量, 自店销量, 自店市占比 |
| 新品PLP数据 | 广告campaign级明细（98行×39列） | 活动, SKU, PLP开启日期, 曝光/点击/售出/花费/广告销售额/ACOS/ACOAS |

## 二、数据处理管线

```
新品月报数据.xlsx
    ↓ 拖入浏览器
月报_5月.html
    ├── parseMonthlyData() → 解析6个Sheet → DATA对象
    ├── 8个Tab渲染（Chart.js 4.4.0）
    └── exportXLSX() → 10 Sheet Excel导出
```

### KPI 卡片小字规范
- 统一格式：`上月 XXX | 环比 +/-XX%`
- 市占比环比用百分点差（pp），销量/销售额用百分比（%）
- 环比计算函数 `hb(v)`：输入为小数比率，内部×100显示

### 表格样式
- `.data-table`: 12px字体，表头 `#f5f7ff` 背景，hover行高亮
- `.table-scroll`: max-height 400px，overflow auto，表头 sticky 冻结
- `.kpi-card`: 白色卡片，圆角8px，阴影，左侧4px彩色边框

## 五、市占比计算规则

```
市占比 = (Σ月自身销量 + ΣPW自店) ÷ (Σ月自身销量 + Σ月对手销量 + ΣPW自店 + ΣPW对手)
```

- 3-4月上架SKU：使用实际月对手数据
- 5月上架SKU：对手数据未出，使用PW爬虫数据补充
- 5月时KPI hint标注 "5月为PW数据"
- PW数据源：5.31PW Sheet，col2=对手销量，col3=自店销量，col4=自店市占比

## 六、月市容量计算规则

```
月市容量 = 自身月销量 + 对手月销量
```

- 有当月对手数据 → 用当月
- 无当月对手 + 非当月新上架 → 用上月对手
- 当月新上架 → 只有自身销量

分档：≤6低 / 7-15中 / ≥16高

## 七、指标命名规范（强制）

- KPI标签宁简勿繁：`市占比` 不是 `加权市占`
- 计算公式放hint副标题，不放主标签
- PW数据必须标注 `PW市占(当月参考)`
- 所有市占计算用标准汇总公式：`总销量÷(总销量+总对手)`，不做SKU级加权
