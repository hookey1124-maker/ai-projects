## 五、指标计算规则

### 市占比（全月统一公式）
```
市占比 = 总自身销量 ÷ (总自身销量 + 总对手销量)
```
所有月份均使用实际月度对手数据，不做 PW 补充。

### 月市容量
```
市容量 = 自身月销量 + 对手月销量
```
分档：≤6 低 / 7-15 中 / ≥16 高

### 出单率（两种）
- 总出单率 = (有对手出单 + 无对手出单) ÷ 总SKU数
- 有对手出单率 = 有对手出单 ÷ (有对手出单 + 有对手未出单)

### 广告标签（PLP明细表）
| 标签 | 条件 |
|------|------|
| 正常 | 其余 |
| 高ACOS | ACOS > 15% |
| 花费不出单 | 花费 > $10 且 售出=0 |
| 低花费 | 花费 < $5 |

## 六、指标命名规范（强制）

- KPI 标签宁简勿繁：`市占比` 不是 `加权市占`
- 计算公式放 hint，不放主标签
- 所有市占用标准汇总公式，不做 SKU 级加权
- 环比统一用 `hb()` 调用，输入小数比率

## 九、常见问题

1. **Tab渲染错误**：检查 Console 红字报错，多数是变量未声明（如某 Tab 用了另一个 Tab 的局部变量）
2. **数据全空**：检查 Sheet 名称是否匹配（中文编码），列索引是否正确
3. **环比值为 0**：`hb()` 输入为小数比率，不是百分数；0 值会显示 `-`
4. **onclick 失效**：inline 事件中调用的函数必须挂在 `DATA` 全局对象上
5. **DOM 错乱**：检查各 Tab 的 `<div>` 配对，Tab 之间必须是兄弟节点不能嵌套
