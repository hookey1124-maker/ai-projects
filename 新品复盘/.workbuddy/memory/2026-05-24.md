### JS 避坑记录
- **Python raw string `r'''...'''` 中的单引号转义**：`\'` 在 raw string 中会变成字面量 `\'` 输出到 JS，导致语法错误。修复方案：使用命名函数 + `onclick="funcName()"` 替代内联复杂 JS 字符串
- **多选复选框 onClickOutside**：使用 `document.addEventListener('click', ...)` 统一监听，判断 `e.target` 是否在 dropdown 或 label 内
