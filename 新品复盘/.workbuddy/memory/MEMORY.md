## 统计规则（2026-04-28确立 - 用户确认）

### 核心口径
1. **品线/分析人"上架产品数"**：只统计周期内实际上架的新品
   - "4.16-4.22上架产品数"= 该周期内上架的新品数
   - 不是"截止4.22的累计SKU数"
   - 4.16-4.22实际上架=0（无产品在4.16-4.22期间上架）

2. **新品出单情况（有对手口径）**：
   - "有对手"= 对手销量 > 0的SKU
   - 4.15有对手=52（源数据验证✓）
   - 4.22有对手=70（源数据验证✓）
   - 未出单/8日外/8日内：需要从8日出单情况列提取，源数据该列为Y/N/未出单/未上架

3. **分析及时率**：
   - 截止某日期的统计口径= 所有上架日期<=截止日期的SKU
   - iv=0：8日内新品无分析
   - 0<iv≤7：及时分析产品
   - iv>7：超7日未分析产品
   - iv=未上架：当日上架未记录（不计入总数）

4. **新品未出单原因**：
   - 按4.22市场状态字段分布
   - 竞争无优势/无市场/站内无价格优势/站外出单

5. **拓展类型出单SKU数**：
   - 8日出单情况列中不为"未出单"的所有值

## PLP ACOAS计算规则（2026-05-13新增）

### 定义
- **ACOAS** = 广告花费 / 总销售额
- 计算公式：`ACOAS = PLP广告花费合计 / 该SKU 4.30-5.6周期总销售额`

### 数据来源
- 广告花费：从 `PLP明细` Sheet 按SKU汇总同周期花费
- 总销售额：从 `四三数据累计` Sheet 的 `4.30-5.6销售额` 列（索引25）
- 同一SKU有多个广告活动时，需合计所有活动的花费后再除以总销售额
