# 工作区 — VS Code 多根工作区

> 正文必须用中文。本文件是项目迁移记忆文件。

## 项目定位

VS Code 工作区配置文件，用于跨项目联动。名称为"模板提取"。

## 内容

仅一个文件：`模板提取.code-workspace` (252 B)

## 引用的项目

```json
{
  "folders": [
    { "path": "." },                    // 工作区自身
    { "path": "../新品复盘" },          // 新品复盘
    { "path": "../三部周报v1" },        // 三部周报v1
    { "path": "../对手商品爬虫" },      // 路径可能损坏（编码问题）
    { "path": "../agent升级计划" }      // agent升级计划
  ]
}
```

## 注意事项

- 本目录不是真正的"项目"，只是 VS Code 配置归档
- `../对手商品爬虫` 路径可能因中文编码问题而无法解析
- 工作区名"模板提取"暗示用于跨项目模板标准化
- 到新电脑后，文件夹路径变化需手动修改 `.code-workspace` 文件
